What is a html file?

Originally a language invented to improve the sharing of documents amongst CERN researchers, HTML (Hypertext Markup Language) is the foundation of individual web pages. HTML tags input into a text-editor define a number of attributes—bold or italicized fonts are two simple examples; these HTML tags are read by a browser and interpreted as the web sites which with we are all familiar. A web site’s HTML can be viewed through a browser’s “View Source” option. HTML tags have two easily distinguishable parts, the start tag and the end tag. Links, images, text, or other data are programmed between the start and end tags.  Other examples of source code files are C, FOR and H files.
